# 24-card-game-solver

Available online at https://acs3ss.github.io/24-card-game-solver! 

## Python usage

```python
python play24.py
```

Two game modes:
- Play 24 with randomly generated cards 
- Check solutions

## Web development

1. `npm install`
1. `npm run serve`
1. Open http://localhost:8080
